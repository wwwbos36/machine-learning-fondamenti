# Fatal Signal — Higgsfield Base Prompts

Prompt base riutilizzabili. Stile fisso: `deep navy and amber palette, 21:9,
anamorphic, film grain, cinematic`. Aggiungere sempre questo suffisso comune
quando non già presente. Mai persone reali, loghi reali o eventi storici via AI
(quelli vanno da stock/archivio).

## INTRO FIRMA (3–5s, ricorrente in OGNI video)
```
cinematic slow dolly-in through a dark abstract corporate hallway,
volumetric light, deep navy and amber palette, 21:9, anamorphic,
film grain, no text, no people
```

## ASCESA
```
a glowing modern tech campus rising at dawn, hopeful warm amber light,
slow crane up, wide angle, cinematic, film grain
```

## ERRORE / TENSIONE
```
a single cracked glass office window overlooking a dark city at night,
slow push-in, ominous teal-orange grade, anamorphic
```

## CROLLO
```
empty abandoned open-plan office, dust particles in light beams,
slow tracking shot, melancholic, desaturated, cinematic
```

## TESI FINALE
```
lone chess king falling in slow motion on dark reflective surface,
dramatic amber rim light, macro lens, black background, film grain
```

## TRANSIZIONE TRA CAPITOLI
```
extreme close-up of a glowing circuit board, slow pull-back reveal,
deep blue light, macro, anamorphic flare
```

## Regole d'uso
- MAX 40% delle scene totali da Higgsfield (il resto stock + archivio reale).
- Clip Higgsfield tipiche 5–15s → stitch in DaVinci per durate maggiori.
- Mantenere coerenza: stessa palette/lens/grana in ogni prompt.
- Ogni nuova scena utile va salvata in `visual-library.md` per il riuso.
- Generazione via MCP Higgsfield: `generate_video` (image-to-video o text-to-video),
  `generate_image` per keyframe, `motion_control`/`presets_show` per i camera-move.
