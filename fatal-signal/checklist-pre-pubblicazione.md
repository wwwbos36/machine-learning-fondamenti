# Fatal Signal — Checklist Pre-Pubblicazione

Controlli umani OBBLIGATORI prima di pubblicare ogni episodio.

```
[ ] Fatti verificati su fonte primaria (no allucinazioni)
[ ] "The Fatal Flaw" è identificato chiaramente nello script
[ ] Video editorialmente distinto dagli episodi precedenti
[ ] Struttura thumbnail diversa dall'episodio precedente
[ ] Script letto ad alta voce (test ritmo)
[ ] Mix AI + stock reale verificato (no tutto-Higgsfield, ≤40% AI)
[ ] Disclosure AI inclusa nella descrizione
[ ] Anti-duplicazione log aggiornato
[ ] Audio normalizzato a -16 LUFS
[ ] Titolo + thumbnail superano il test "cliccherei?"
[ ] Capitoli/timestamp corretti
```

## Test policy "inauthentic" (luglio 2025)
Lo spettatore medio capisce chiaramente che questo video è diverso dagli altri del canale?
→ Se NO, riscrivere prima di pubblicare.
