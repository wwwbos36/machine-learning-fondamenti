# 💰 MODELLO FINANZIARIO E PRICING

## Formula Prezzo

```
Prezzo = (Costo Base + Spedizione + Fee Piattaforma + Buffer Resi) ÷ (1 − Margine%)

Fee Etsy ≈ 12% (6.5% TX + €0.20 listing + ~3% processing)
Fee Shopify ≈ 3% (solo payment 2.9% + €0.30)
Buffer resi ≈ 2-3% del costo base
```

## Break-even Mensile (esempio)

```
Costi fissi mensili stimati:
- Shopify base:        €27
- App (Klaviyo free + Loox €10): €10
- Tool trend (eRank):  €10
- Gelato+ (opzionale): €15
─────────────────────────────
TOTALE FISSO:          ~€62/mese

Break-even = €62 ÷ €11.47 profitto/tee ≈ 6 t-shirt/mese
```
